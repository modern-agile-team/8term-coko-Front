## 📌 커밋 메시지 컨벤션

> **git commit -m `"✨ Feature(#123): 기능 추가"`**

| Type             | Description       |
| ---------------- | ----------------- |
| 🌏 Deploy        | 배포 관련         |
| ⚙ Setting        | 개발 환경 설정    |
| 🐞 BugFix        | 버그 수정         |
| 💻 CrossBrowsing | 브라우저 호환성   |
| 📃 Docs          | 문서 작성 및 수정 |
| ✨ Feature       | 기능 추가         |
| 📬 API           | API 통신          |
| 📝 Modify        | 기능, 코드 수정   |
| 🔨 Refactor      | 코드 리팩토링     |
| 🎨 Publish       | UI/디자인 작업    |
